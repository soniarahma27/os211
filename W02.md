## Top 10 List of Week 02

### 1. [ENKRIPSI SIMETRIS DAN ASIMETRIS](http://webpintar.blogspot.com/2018/02/pengetian-simetris-dan-asimetris.html)
Artikel ini berisi penjelasan tentang perbedaan enkripsi simetris dan asimetris. Disini juga dijelaskan kelebihan dan kekurangannya masing-masing, disertai contoh.
### 2. [Learn C Language](https://www.learn-c.org/)
Situs ini berisi tutorial belajar C language dari basic. Tutorialnya sangat mudah dipahami dan disertai latihan.
### 3. [Protection and Security in Operating System](https://www.tutorialspoint.com/Protection-and-Security-in-Operating-System)
Artikel ini berisi penjelasan tentang jenis-jenis ancaman terhadap keamanan sistem operasi dan perlindungannya.
### 4. [Difference Between Security and Protection](https://www.geeksforgeeks.org/difference-between-security-and-protection/?ref=rp)
Artikel ini menjelaskan tentang perbedaan antara security dan protection.
### 5. [Difference between Antivirus and Internet Security](https://www.geeksforgeeks.org/difference-between-antivirus-and-internet-security/?ref=rp)
Situs ini berisi penjelasan tentang perbedaan antara antivirus dan internet security.
### 6. [Protection Ring](https://www.geeksforgeeks.org/protection-ring/)
Situs ini berisi penjelasan tentang protection ring. Penjelasannya lengkap dan sangat mudah dipahami.
### 7. [Access matrix in Operating System](https://www.geeksforgeeks.org/access-matrix-in-operating-system/)
Artikel ini berisi penjelasan tentang Access Matrix pada sistem operasi.
### 8. [Public and Private Key Encryption](https://www.preveil.com/blog/public-and-private-key/)
Artikel ini berisi penjelasan tentang public dan private key encryption. Penjelasannya lengkap dan mudah dipahami, disertai contoh penerapannya.
### 9. [Why Linux is better than Windows or macOS for security](https://www.computerworld.com/article/3252823/why-linux-is-better-than-windows-or-macos-for-security.html)
Artikel ini menjelaskan alasan bahwa linux lebih baik dari Windows maupun macOS dalam hal security.
### 10. [How Security System Should Evolve to Handle Cyber Security Threats and Vulnerabilities?](https://www.geeksforgeeks.org/how-security-system-should-evolve-to-handle-cyber-security-threats-and-vulnerabilities/?ref=rp)
Artikel ini menjelaskan tentang bagaimana seharusnya security system berkembang dalam menangani ancaman cyber security.
