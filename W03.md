## Top 10 List of Week 02

### 1. [File Systems in Operating System](https://www.guru99.com/file-systems-operating-system.html)
Artikel ini menjelaskan tentang file system pada Sistem operasi. Artikel ini sangat menarik karena penjelasannya jelas dan lebih ringkas sehingga mudah dipahami.
### 2. [File Management System in Operating System](https://medium.com/@princeabhishek410/understanding-file-management-system-in-operating-system-4c7fbfc306f2)
Artikel ini menjelaskan tentang file management system pada sistem operasi. 
### 3. [Mount and Unmount File System/Partition in Linux](https://www.2daygeek.com/mount-unmount-file-system-partition-in-linux/)
Web ini berisi penjelasan tentang mount dan unmount file system di Linux. Penjelasannya sangat jelas dan mudah dimengerti, disertai dengan contoh.
### 4. [File System Implementation in Operating System](https://www.geeksforgeeks.org/file-system-implementation-in-operating-system/)
Artikel ini menjelaskan tentang implementasi file system pada sistem operasi. Implementasi file system menggunakan tipe data struktur on-disk dan in-memory. Penjelasan dari artikel ini sangat mudah untuk dipahami.
### 5. [Linux File Hierarchy Structure](https://www.geeksforgeeks.org/linux-file-hierarchy-structure/)
Artikel ini berisi penjelasan tentang File System Hierarchy Standard (FHS) pada Linux. FHS adalah sebuah susunan direktori dan sub direktori pada Linux, dimulai dari / dan berisi direktori-direktori di dalamnya. Penjelasannya sangat mudah dipahami dan disertai gambar.
### 6. [File System Internal](https://www.netbsd.org/docs/internals/en/chap-file-system.html)
Web ini berisi penjelasan tentang file system internal. Penjelasan dari situs ini sangat lengkap dan mudah dipahami.
### 7. [Network File System (NFS)](https://www.geeksforgeeks.org/network-file-system-nfs/)
Artikel ini berisi penjelasan tentang Network File System (NFS). NFS dikelola oleh Sun Microsystems. Network File System/NFS adalah sebuah kumpulan protokol yang digunakan untuk mengakses beberapa sistem berkas melalui jaringan.
### 8. [Virtual filesystems in Linux: Why we need them and how they work](https://opensource.com/article/19/3/virtual-filesystems-linux)
Artikel ini berisi penjelasan tentang mengapa Virtual File System itu diperlukan. Selain itu, artikel ini juga menjelaskan bagaimana Virtual File System bekerja. Penjelasannya sangat menarik dan mudah dipahami.
### 9. [Log-Structured File System (LFS)](https://www.geeksforgeeks.org/log-structured-file-system-lfs/)
Artikel ini menjelaskan tentang Log Structured File System (LFS). Log Structured File System (LFS) adalah sistem file di mana data dan metadata ditulis secara berurutan ke buffer melingkar, yang disebut log.
### 10. [Consistency Semantics for file sharing](https://www.geeksforgeeks.org/consistency-semantics-for-file-sharing/)
Artikel ini menjelaskan tentang Consistency Semantic pada Sistem Operasi. Consistency Semantics adalah konsep yang digunakan oleh pengguna untuk memeriksa sistem file yang mendukung berbagi file di sistem.
